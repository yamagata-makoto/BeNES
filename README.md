BeNES
======
### Famicom(Nintendo Entertaiment System) Emulator

Howto build
---
for Haiku32 or BeOS
```
$ cd BeNES
$ make -f makefiles/Makefile.BeOS
```

BeNES keys
---
- Up/Down/Left/Right: arrows
- A button: Z, alt
- B button: X, left shift, ctrl
- START: enter
- SELECT: space
