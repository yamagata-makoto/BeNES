# Famicom Emulator for BeOS/Haiku
since (C) 2000 makoto
---

Howto build
---
for Haiku
```
$ git https://github.com/yamagata-makoto/SEAL.git
$ cd SEAL/src
$ make haiku
$ cd ../..
$ make
```

BeNES keys
---
Up/Down/Left/Right: arrows
A button: Z, alt
B button: X, left shift, ctrl
START: enter
SELECT: space
